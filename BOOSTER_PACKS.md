# StartX · Booster Packs（卡包全集 · 各阶段）

> 基于 `Stacklands_like_Enterprise_Resource_Management_GDD_CN.pdf` 第 6.1 节整理。
> 规则：每包 3–5 张卡，每个 slot 有独立概率池；**Idea 卡一次性掉落**，已拥有则替换为默认资源；和平/教学模式下敌人 slot 自动替换为默认资源。
> 「切片」标注是否已在当前 MVP（`game/data/packs.json`）中实现。

---

## 阶段总览

| 阶段 | 卡包 | 解锁节奏 |
|------|------|----------|
| 起步 | 创业起步包 / 基础运营包 | 开局～首次卖卡 |
| 产品 | 产品研发包 / 销售增长包 | 首个 PRD ～ 首个商机 |
| 经营 | 中后台包 / 交付管理包 | 现金达标 ～ 首个合同 |
| 资本 | 融资包 | 连续 2 月现金流为正 |
| 扩张 | 产业资源包 / 自动化包 / 危机管理包 | 产业合作 / 3设施 / 首次重大风险 |
| 终局 | 海外扩张包 / 上市准备包 | 许可证+3合同 / 年收入100或B轮 |

---

## 阶段 1 · 起步

### 🟢 A New Venture 创业起步包
- **价格**：0（开局免费）｜**张数**：4｜**解锁**：开局自动获得
- **卡池**：创始人、现金、办公室、市场线索池
- **功能**：提供最初的人、钱、场地、线索源，能立刻开始第一条生产链。
- **切片**：🔶 等效于开局直接发的 6 张起手牌（创始人/销售/产品经理/线索池/客户需求/办公室）。

### 🟢 Humble Operations 基础运营包
- **价格**：现金 3｜**张数**：3–4｜**解锁**：完成卖出一张卡
- **卡池**：现金、线索、客户需求、办公室、`Idea: 招聘`
- **功能**：教会"卖卡→买包"的货币循环，补充早期线索与需求，解锁招聘配方。

---

## 阶段 2 · 产品

### 🔵 Product Lab 产品研发包
- **价格**：现金 5｜**张数**：3–5｜**解锁**：完成第一个 PRD
- **卡池**：工程师、产品经理、技术难题、PRD、代码、`Idea: MVP`
- **功能**：搭起研发链（技术难题→代码→原型），解锁 MVP 开发路线。

### 🟠 Sales Growth 销售增长包
- **价格**：现金 6｜**张数**：3–5｜**解锁**：完成第一个商机
- **卡池**：销售、市场经理、线索池、商机、CRM、`Idea: 企业销售`
- **功能**：放大获客与转化能力，CRM 自动聚合线索，解锁企业销售。
- **切片**：✅ 已实现（slot 权重池：销售专员/线索/商机/线索池/客户需求/CRM/产品经理）。

---

## 阶段 3 · 经营

### ⚪ Back Office 中后台包
- **价格**：现金 7｜**张数**：3–5｜**解锁**：现金余额达到 10
- **卡池**：财务、法务、HR、制度、财务系统、法务档案
- **功能**：补齐财/法/人三大中后台，建立制度降低风险、可视化成本。

### 🟤 Delivery 交付管理包
- **价格**：现金 8｜**张数**：3–5｜**解锁**：签署第一个合同
- **卡池**：运营主管、交付中心、客户成功、合同、回款
- **功能**：把合同变成稳定回款，交付中心提速、客户成功保续约。

---

## 阶段 4 · 资本

### 🟣 Capital 融资包
- **价格**：现金 10｜**张数**：3–5｜**解锁**：连续 2 月现金流为正
- **卡池**：IR、投资人网络、财报、数据室、TS、`Idea: 尽调`
- **功能**：打通融资链（投资人会议→TS→融资款），获得大额资金扩张。

---

## 阶段 5 · 扩张

### 🏛️ Industry 产业资源包
- **价格**：现金 12｜**张数**：3–5｜**解锁**：完成产业合作任务
- **卡池**：政府窗口、政策窗口、研究院、许可证线索、补贴
- **功能**：进入政企/产业市场，获取补贴与许可证线索。

### ⚙️ Automation 自动化包
- **价格**：现金 14｜**张数**：3–5｜**解锁**：拥有 3 个设施
- **卡池**：自动化工作台、流程制度、AI Copilot、数据湖
- **功能**：把常用产出自动归堆、降低生产时间，缓解棋盘混乱。

### 🛡️ Crisis 危机管理包
- **价格**：现金 8｜**张数**：3–5｜**解锁**：遭遇第一次重大风险后
- **卡池**：法务、风控、保险、风险委员会、反舆情策略
- **功能**：系统性降低高阶风险触发率与伤害，专治舆情/监管/纠纷。

---

## 阶段 6 · 终局

### 🌏 Global 海外扩张包
- **价格**：现金 18｜**张数**：3–5｜**解锁**：取得许可证并完成 3 个合同
- **卡池**：海外办公室、海外线索、本地合规、汇率风险、渠道伙伴
- **功能**：开启海外棋盘与多货币经营，引入汇率与本地合规变量。

### 📈 IPO 上市准备包
- **价格**：现金 25｜**张数**：3–5｜**解锁**：年度收入达到 100 或完成 B 轮
- **卡池**：审计、券商、律师、IPO 材料、内控体系
- **功能**：集齐上市申报要件，冲击终局"上市被否"Boss。

## 卡包数据结构（实现参考）

```ts
type PackDefinition = {
  id: string;
  name: string;
  price: { currency: 'cash'|'subsidy'|'foreign_cash'|'valuation'; amount: number };
  minCards: number; maxCards: number;
  unlockQuestId?: string;
  slots: PackSlot[];
};
type PackSlot = {
  slotIndex: number;
  options: { cardId: string; weight: number; once?: boolean; replaceWith?: string }[];
  enemyReplacement?: string; // 和平/教学模式
};
```
